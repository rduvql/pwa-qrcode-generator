# Activity Planner

A lightweight PWA qrcode generator

client side only

## Features

- **Qr code generation from url**
- **Client side only**

## Live Demo

[**https://rduvql.github.io**](https://rduvql.github.io)

## Built With

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Vite PWA Plugin

## Installation
```bash
# Install dependencies
pnpm install

# Run development server
pnpm run dev:w

# Build for production
pnpm run build
```
